## Описание (вставить в поле "О заказе")

Создам профессиональный лендинг или одностраничный сайт с нуля — чистый HTML/CSS/JS, без тяжёлых фреймворков.

**Что входит в базовый пакет:**
- Адаптивный дизайн (мобильный + десктоп)
- Тёмная или светлая тема на выбор
- Анимации появления элементов при скролле
- Форма обратной связи или кнопка CTA
- Быстрая загрузка (никаких лишних библиотек)
- Публикация на GitHub Pages (бесплатный хостинг) — по желанию

**Примеры моих работ:**
- Лендинг FocusForge: https://georik015.github.io/focusforge
- Визитка-портфолио: https://georik015.github.io/focusforge/vizitka.html

**Буду рад взяться за:**
- Лендинг для продукта / стартапа
- Страница-портфолио или резюме
- Страница для мероприятия / конкурса
- Конвертация Figma-макета в HTML
