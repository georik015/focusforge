# Визитка-карточка — Тёмная тема (HTML шаблон)

Профессиональная цифровая визитка в тёмном glassmorphism-стиле.
Один HTML-файл, без зависимостей — открывается в любом браузере.

## Что внутри

- Split-layout: фото слева, информация справа
- Адаптивный дизайн (мобильный + десктоп)
- Тёмная тема: фиолетовый + циановый градиент
- Анимированные accent-линии
- Кнопки соцсетей: Telegram, GitHub, VK, Email
- CTA-кнопка "Написать мне"
- Никаких фреймворков и зависимостей

## Как настроить (3 минуты)

Открой `vizitka.html` в любом редакторе и замени:

| Что найти                        | На что заменить              |
|----------------------------------|------------------------------|
| `Георгий Созанов`                | Твоё имя                     |
| `Программист`                    | Твоя специальность           |
| `Пишу код, решаю задачи...`      | Твоё описание (1-2 строки)   |
| `https://t.me/SozanovGeorik`     | Твой Telegram                |
| `https://github.com/georik015`   | Твой GitHub                  |
| `mailto:sozanov@example.com`     | Твой Email                   |
| `<img src="data:...`             | Тег с твоим фото             |

### Как заменить фото

```html
<!-- Вместо тега <img src="data:image/..."> вставь: -->
<img src="photo.jpg" alt="Фото" />
```

Положи `photo.jpg` рядом с HTML-файлом.

### Как изменить цвета (в начале файла)

```css
:root {
  --accent1: #7c3aed;  /* фиолетовый → можно менять */
  --accent2: #06b6d4;  /* циановый → можно менять */
  --bg: #05030e;       /* фон */
}
```
